> ECMAScript 5/6/7 [compatibility tables](https://github.com/kangax/compat-table) CLI

## Install

```
$ npm install --global compat-cli
```

## Usage

```
$ compat-cli --help

  Usage
    $ compat-cli <name> [options]

  Options
    --beast-mode    Put additional 45lbs plates per side if doing back squats

  Examples
    $ compat-cli rest
```
